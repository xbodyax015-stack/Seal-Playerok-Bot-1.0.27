## 🖥️ Установка на Windows

1. Скачайте последнюю версию и распакуйте
2. Установите [**`Python 3.12`**](https://www.python.org/downloads/release/python-31210/) ⚠️ **Важно: именно эта версия!** 
3. Запустите `Setup.bat` для установки зависимостей
4. Запустите `start.bat`
5. Следуйте инструкциям для настройки бота

## 🐧 Установка на Linux/VDS (Ubuntu 20.04+) НО желательно Ubuntu 24.04

### 🛒 Где купить VDS?

Рекомендуем: **[FirstByte VDS](https://firstbyte.ru/?from=192586)** — недорогие и надёжные сервера.

> ⚠️ **Требования:** Ubuntu 20.04+ но лучше Ubuntu 24.04 - загрузка пройдёт быстрее

### 📋 Пошаговая инструкция для новичков

<details>
<summary><b>🔰 Полная инструкция (нажми чтобы раскрыть)</b></summary>

#### Шаг 1: Купи VDS сервер

1. Зайди на сайт хостинга (например [FirstByte](https://firstbyte.ru/?from=192586))
2. Выбери тариф (за 75 рублёв там есть или за 200 в месяц)
3. Выбери ОС: **Ubuntu 22.04** или **Ubuntu 24.04** (рекомендуется)
4. После оплаты в инструкции к своему сервкру на сайте находишь: **IP, логин (root), пароль**

#### Шаг 2: Подключись к серверу
Рекомендуем универсальный SSH-клиент - [Termius](https://www.termius.com/download/windows) можете использовать любой дргугой клиент 
1. Скачиваете клинет
2. Подключаетесь к своему серверу по IP и User с паролем


#### Шаг 3: Установи бота (одна команда!)

Скопируй и вставь эту команду:

```bash
curl -sSL https://raw.githubusercontent.com/leizov/Seal-Playerok-Bot/main/install.sh | sudo bash
```

или если не сработало через wget:

```bash
wget -qO- https://raw.githubusercontent.com/leizov/Seal-Playerok-Bot/main/install.sh | sudo bash
```

#### Шаг 4: Следуй инструкциям установщика

1. **Введи имя пользователя** — просто нажми Enter для "sealbot" по умолчанию
2. **Дождись установки** — займёт 2-5 минут
3. **Введи токены** когда бот попросит:
   - Токен Playerok (из настроек аккаунта Playerok)
   - Токен Telegram бота (создай через [@BotFather](https://t.me/BotFather))
4. **Нажми Ctrl+C** когда увидишь главное меню бота
5. **Готово!** Бот работает в фоне

#### Шаг 5: Настройка автозапуска и запуск бота

После завершения настройки выполни следующие шаги:

1. **Отключись от сессии** — нажми `Ctrl+C` чтобы выйти из интерактивного режима
2. **Включи автозапуск** — чтобы бот запускался при перезагрузке сервера:
   ```bash
   seal-sealbot enable
   ```
3. **Запусти бота**:
   ```bash
   seal-sealbot start
   ```
4. **Проверь статус**:
   ```bash
   seal-sealbot status
   ```

> 💡 Замени `sealbot` на имя пользователя, которое ты ввёл при установке!

</details>

### 🎮 Команды управления (супер фича для вдс юзеров)

После установки используй команду `seal-ИМЯ_ПОЛЬЗОВАТЕЛЯ`:

> 💡 Если при установке ввёл "sealbot" — команда будет `seal-sealbot`  
> Если ввёл "mybot" — команда будет `seal-mybot`

| Команда | Описание |
|---------|----------|
| `seal-xxx start` | 🚀 Запустить бота |
| `seal-xxx stop` | 🛑 Остановить бота |
| `seal-xxx restart` | 🔄 Перезапустить бота |
| `seal-xxx status` | 📊 Статус бота |
| `seal-xxx logs` | 📋 Логи в реальном времени (Ctrl+C выход) |
| `seal-xxx logs100` | 📋 Последние 100 строк логов |
| `seal-xxx update` | 🔄 Обновить бота |
| `seal-xxx enable` | ✅ Включить автозапуск |
| `seal-xxx disable` | ❌ Отключить автозапуск |

> Ещё раз seal-xxx это seal-ИМЯ_ПОЛЬЗОВАТЕЛЯ
> 📌 **Несколько ботов:** Можно установить несколько ботов на один сервер с разными именами пользователей!
